Dynamic Currency Conversion

Objective: Multi-currency conversions on sales indicators via slicer rates.

Converted Sales = 
VAR SelectedRate = 
  SELECTEDVALUE(Dim_ExchangeRates[Rate], 1)
RETURN 
  SUM(Fact_Sales[SalesAmount]) * SelectedRate
Logic: Evaluates currency slicer context, returning exchange rate multipliers.

30 Day Moving Average

Objective: Smooth fluctuation variations representing daily sales flow.

Sales 30D Moving Avg = 
AVERAGEX(
  DATESINPERIOD(
    Dim_Dates[Date],
    LASTDATE(Dim_Dates[Date]),
    -30,
    DAY
  ),
  [Total Sales]
)
Logic: Calculates 30-day window averages shifting current date constraints.
